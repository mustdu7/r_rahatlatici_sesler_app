# R: Rahatlatıcı Sesler

Doğanın huzur veren seslerini bir araya getiren uyku ve meditasyon uygulaması.
Kendi ses karışımını oluştur, zamanlayıcı kur ve rahatça uyu.

## Özellikler
- 13 farklı doğa sesi
- Özelleştirilebilir ses karışımı
- Uyku zamanlayıcısı
- Ana ekran widget desteği
- Kilit ekranı kontrolü
